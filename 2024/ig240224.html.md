## macOS 用 Windows 風味スイッチャー - AltTab

私用 MacBook Air に AltTab をインストールしました。

- https://alt-tab-macos.netlify.app/

Command + Tab を AltTab に上書き割り当てすると Windows 的にアプリをスイッチングできます。
