Eclipse 3.3 日本語化言語パック(サードパーティ版) β版 リリース

blanco Frameworkは、Eclipse 3.3 日本語化言語パック(サードパーティ版) β版 をリリースしました。

  • blancoNLpackGenerator - Eclipse 3.3 日本語化言語パック(サードパーティ版)

Eclipse Classic (SDK), Europa Fall Distros, PHP Development (PDT) などのための言語パック(サードパーティ版)がダウンロードできるようになっています。

※この件については オープンソースカンファレンス 2008 Tokyo/Spring 2008.02.29(金) 「Eclipse日本語言語パック 開発プロジェクト」において詳細な説明がおこなわれる予定です。(なお、02/28 17:06時点では 残席が少なくなりつつあります)
