## 製作中のオーダーメイド・ビオラの概要メモ

いま製作してもらい中のオーダーメイド・ビオラの概要をメモしておきます。ようやく表板のニス塗りが佳境に入り、もうすぐネックを取り付けるという段取りです。製造着手してから1年半ほど経っています。年内に完成する可能性があります。

### 1. 目指す楽器像

- 1850 年頃の“モダン”ビオラを再現。
- オールドの華やかさでも、現代コンテンポラリーの過剰なパワーでもなく、内声楽器らしい控えめな響きと誠実さを大切にする。
- チェロには近づけず、大きなバイオリンを基本とする。

### 2. 基本仕様

| 項目      | 内容                                     |
| ------- | -------------------------------------- |
| 型紙      | Stradivarius 171X 年型の外観を基にした約 41 cm ボディ   |
| 表板      | スプルース・ブックマッチ、極厚指向                      |
| 裏板      | バルカン産メイプル 1 枚板、極厚指向                    |
| アーチ高    | 表 17.5 mm／裏 16.5 mm（ハイアーチ、彫り深めで立体感を強調） |
| 横板高さ    | ネック側 34 mm／テール側 35 mm（やや低リブで抱えやすさを確保）    |
| ブロック    | ネック・テールとも大型で堅牢 ― 鳴り過ぎを抑え輪郭を明確化         |
| ネックヘッド幅 | 46 mm（ポジション感の安定を重視）                    |
| ネック長    | 146 mm                    |
| 駒       | Despiau                                |
| ニス       | 未精製シェラック（waxed）                                |

### 3. 設計哲学と音響方針

- 控えめな常鳴（ジョウメイ）
    - ふだんは落ち着き、必要なときにだけ豊かに語る。
- 厚板 × ハイアーチ × 低リブ
    - 極厚板で芯と持続力を確保。
    - ハイアーチで空間容積と豊かな倍音を補完。
    - 低リブにより 41 cm サイズでも身体負担を軽減。
- 大型ブロックによる“鳴りの制御”
    - 過度な余韻を抑え、フォーカスした音像を実現。
    - 初期はややタイトだが、弾き込む程に開く設計。
- 経年で育つポテンシャル
    - 厚板と堅牢構造が、年月の振動でゆっくり開き、“熟成する音”を念頭に置く。

### 4. 奏者へのメリット

- 持ちやすさ：薄い胴体と低リブで長時間演奏の負担を軽減。(ただし重量は大きくなる見込み)
- 音色の個性：ビオラらしい中域の暖かさが際立ち、弦楽四重奏で内声を自然に支える。
- 成長する楽器：弾き込みとともに深みが増し、演奏者と一緒に歳を重ねる楽しみ。（書いてて気づいた。私あと何年楽器弾けるかな?）
