## 複数の wav ファイルを1つの wav ファイルに連結

複数のwavファイルを1つに連結する方法をメモします。

### filelist.txt を準備

以下のような内容のテキストファイル `filelist.txt` を作成します。

file ‘250426_092408_TrMic.WAV’
file ‘250426_101043_TrMic.WAV’
file ‘250426_105718_TrMic.WAV’
file ‘250426_114353_TrMic.WAV’

### コマンド

次のffmpegコマンドを実行します。

```
ffmpeg -f concat -safe 0 -i filelist.txt -c copy 250426_TrMic.wav
```

#### 引数の説明

| 引数 | 説明 | 詳細 |
|:-----|:-----|:-----|
| `-f concat` | フォーマット指定 | 入力ファイルの形式として「連結モード（concat）」を使用することを指定する。 |
| `-safe 0` | セーフモード無効化 | ファイルパスの安全性チェックを無効化する。ローカルファイルの自由な指定を許可する。 |
| `-i filelist.txt` | 入力ファイルリスト | 連結するファイル一覧が書かれたリストファイル（ここでは `filelist.txt`）を指定する。 |
| `-c copy` | コーデックコピー | エンコードせずに元のデータをそのままコピーして連結する。変換による劣化がない。 |
| `250426_TrMic.wav` | 出力ファイル名 | 連結後の出力ファイル名。ここでは `250426_TrMic.wav` として保存する。 |
