## PlantUML を使用して UML図を作成する手順メモ

PlantUML を用いて、UML図を作成するときのメモ。この記事はあとで体裁を整えるかも（整えないかも）
Gemin でシーケンス図書く時には PlantUMLを利用しますの。
PlantUMLの入手
wget https://github.com/.../v1.2025.2/plantuml-lgpl-1.2025.2.jar
PlantUMLの実行
java -jar plantuml-lgpl-1.2025.2.jar -tsvg sample.puml
PlantUMLの入力ファイル例
sample.puml
@startuml
!theme sunlust
!$a  = 42
!$ab = "foo1"
!$cd = "foo2"
!$ef = $ab + $cd
!$foo = { "name": "John", "age" : 30 }
Alice -> Bob : $a
Alice -> Bob : $ab
Alice -> Bob : $cd
Alice -> Bob : $ef
Alice -> Bob : Do you know **$foo.name** ?
@enduml
