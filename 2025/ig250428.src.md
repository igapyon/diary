## wav ファイル形式を m4a ファイル形式に変換

wav ファイルを指定時間で切り詰め

```
ffmpeg -ss 165  -t 17500 -i 250427_153333_TrMic.WAV -c copy 250427_153333_TrMic-trim.wav
ffmpeg -ss 165  -t 17500 -i 250427_153333_Tr1.WAV -c copy 250427_153333_Tr1-trim.wav
```


```
ffmpeg -i 250427_153333_TrMic-trim.wav -af loudnorm -ar 44100 -sample_fmt s16 250427_153333_TrMic-loudnorm.wav
ffmpeg -i 250427_153333_Tr1-trim.wav -af loudnorm -ar 44100 -sample_fmt s16 250427_153333_Tr1-loudnorm.wav
```

必要に応じて複数ファイルを連結

```
ffmpeg -loop 1 -framerate 2 -i Mozart_K581.png -i 250427_153333_TrMic-loudnorm.wav -c:v libx264 -preset medium -tune stillimage -crf 18 -c:a copy -shortest -pix_fmt yuv420p 250427_153333_TrMic.mkv
ffmpeg -loop 1 -framerate 2 -i Mozart_K581.png -i 250427_153333_Tr1-loudnorm.wav -c:v libx264 -preset medium -tune stillimage -crf 18 -c:a copy -shortest -pix_fmt yuv420p 250427_153333_Tr1.mkv
```
