## Ubuntu 14.04: S.M.A.R.T

S.M.A.R.T の使い方メモ。


### コマンドのインストール


```sh
sudo apt-get install smartmontools 
```



### ディスク一覧


```sh
sudo lshw -short | grep disk
```



### S.M.A.R.T情報取得


```sh
sudo smartctl --all /dev/sda
```



### テスト（ショート）


```sh
sudo smartctl -t short /dev/sda
```
