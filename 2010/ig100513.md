## Ubuntu: Gnome パネル -> システム モニタの設定

Ubuntu 10.04 の Gnome パネルのシステム モニタの個人的好みの色設定をメモします。

* processor
  
  * #002FFF
    
  * #FE00E1
    
  * #00FAFE
  

  
* memory
  
  * #54EA6A
    
  * #081CF7
    
  * #FE1E00
    
  * #BEC60C
  

  
* network
  
  * #CEFB4F
    
  * #FB7B20
  

  
* swap
  
  * #ACC200
  

  
* load
  
  * #018564
  

  
* hdd
  
  * #36CD36
    
  * #B9241D
  

少しずつ改良しようと思います。
