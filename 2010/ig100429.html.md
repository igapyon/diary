## Ubuntu: 実行中のプロセスの優先度を変更する方法

実行中プロセスの優先度を変更するには renice コマンドをもちいます。
実行中のプロセスの優先度を変更する方法

      
```
renice 優先度 -p プロセスID
```

      

優先度ですが、値が大きいほど実行優先度が下がります。優先度を変えた後は、top コマンドで確認することが多いです。
