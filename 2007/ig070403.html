blancoStringConverterにRuby, Python対応を追加開始

blancoStringConverter に Ruby, Python ソースコードの自動生成対応を追加しました。Excelブックの情報を入力として Ruby, Python ソースコードを自動生成することができます。これで対応するソースコード自動生成言語は7言語になります。現在 一通りの実装が終わり、アルファ版のステータスとして公開しています。

2007.04.08追記
