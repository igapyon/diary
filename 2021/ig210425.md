## Oiyokan OData v4 server (provider) SDK

ここ最近の土日で、`Oiyokan` という趣味の OData v4 server を作ってます。

- Oiyokan OData v4 server (provider) SDK
    - https://github.com/igapyon/oiyokan

`Oiyokan` を利用すると、Postgres や MySQL といった RDB を OData v4 server (provider) として OData (REST/JSON) アクセスを可能します。
内部の技術的には、Apache Olingo, Spring Boot, JDBC Driver, h2 database を利用してこれを実現しています。
