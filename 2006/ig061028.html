blancoCgの VB.NET , PHP対応の検討

既に対応済みの Java / C#.NET / JavaScriptソースコード自動生成に加え、blancoCgへの VB.NET および PHP への対応追加を検討しています。VB.NET対応はかねてより検討させられています。PHP対応は、オープンソースカンファレンス2006 Tokyo/Fall の懇親会の席で PHP対応による可能性を指摘され、「PHP対応はニーズが多いのかも」と改めて印象づけられました。2006/06のときのオープンソースカンファレンス.DB でも PHP対応を別の方からすすめられた記憶がよみがえってきました。
