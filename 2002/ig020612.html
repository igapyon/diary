世間のリリース (RSS) 2002
  • SUN Java2 SDK, Standard Edition, 1.4日本語版ドキュメントリリース (O) 2002
    J2SE 1.4 ドキュメントの日本語版リリースです。とても嬉しいです。今までは 1.3用日本語ドキュメントで代用していました。Java屋で1.4をお使いのみなさまには入手をお勧めします。
  • Java Web Services Developer Pack 1.0 リリース (O) 2002
    Sunによるウェブサービスキットのファーストリリースです。JSP標準タグライブラリ(JSTL) 1.0 やAnt, Tomcatなどまで含まれます。結構重要なリリース物かも知れません。便利なパック物であるようにも見えますから。要チェック。
  • Java XML Pack Summer 02 リリース (O) 2002
    JAXP 1.2 などが含まれる様子。これはこれで大変興味有ります。Xerces-Jがどうなったか、などなど。
  • The Java Web Services Tutorial (O) 2002
    ウェブサービスの作成方法に関するイロハ的チュートリアル。XML I/Oのサンプルがあったり、これもまた大変面白そう。
  • Mozilla 1.1 Alpha リリース (O) 2002
    Mozilla作業の方も 早速1.1の方に移ってきている様子です。みなさん頑張ってますね。
