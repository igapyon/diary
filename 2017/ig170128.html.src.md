## [FreeMarker] オブジェクトの boolean 値による if 分岐

[[FreeMarker]] にて、オブジェクトの boolean 値による if 分岐の仕方をメモします。

```xml
${'<'}${'#'}if current.isDiary()${'>'}
  ${'<'}${'#'}-- process --${'>'}
${'<'}/${'#'}if${'>'}
```

気をつける必要のある点は以下です。

* boolean 型の場合は、`current.isDiary()` のように Java 形式記述相当記述が必要そうです。
* 条件分岐にては `{'$'}` は不要です。
