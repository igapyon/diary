## [github] に日記を書いて github 学習のアイデア [igapyonv3]

[[github]] に日記を書いて github の使い方を学習するアイデアを思いつきました。
[[igapyonv3]] を使えば、これを簡易に入門することができることに気が付きました。

[[igapyonv3]] で日記を書くために必要なもの。

* [[github]] のアカウント
  * 日記用の [[gh-pages]] ブランチの作成
* [[git]] コマンドの実行環境
* [[Maven]] の実行環境
  * [[Java]] の実行環境
* 日記に書く内容そのもの
