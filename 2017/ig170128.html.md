## [FreeMarker] オブジェクトの boolean 値による if 分岐

[FreeMarker](../keyword/freemarker.html) にて、オブジェクトの boolean 値による if 分岐の仕方をメモします。

```
<#if current.isDiary()>
  <#-- process -->
</#if>
```

気をつける必要のある点は以下の箇所です。

* boolean 型の場合は、`current.isDiary()` のように Java 形式記述相当記述が必要な様子です。
* 条件分岐にては `{'$'}` は不要です。
