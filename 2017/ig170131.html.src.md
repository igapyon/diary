山本裕介さんの blog 記事を参考に手順をすすめてみます。

* http://samuraism.jp/diary/2012/05/03/1336047480000.html

次にプロダクト登録ですが、groupid などを確定してから申し込みが必要なので、ちょっとここで一時中断。
