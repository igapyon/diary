## 家族用に Microsoft Surface Laptop - 256GB (バーガンディ) を購入

家族用に、Microsoft Surface Laptop - 256GB / Intel Core i5 / 8GB RAM - バーガンディ を購入しました。

購入後に実施したこと。

* Windows 10 Pro にアップグレード (注意: 現在 10 Pro への無料アップグレード期間。購入時は Windows 10 S という版でした)
* Kaspersky Internet Security のインストール
* (未) Microsoft Office のインストール
