## [Mac][shell] .bashrc を有効にする

Mac でホームの `.bashrc` を有効にしてしまうという便利な方法は以下です。

ホームに以下のような内容の `.bash_profile` を作成します。

```sh
if [ -f ~/.bashrc ]; then
    . ~/.bashrc
fi
```

これにより `.bashrc` も動作するようになります。
