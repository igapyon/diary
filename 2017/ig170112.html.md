## [Shell] シェルから内容込みでファイルを作成する方法メモ

bash などの [[unix]] 系 [[shell]] から内容込みでファイルを作成する方法をメモします。

```sh
cat <<MYEOF > /tmp/mysample.sh
#!/bin/bash
echo on

while : 
do
  echo 'hello shell world!'
  sleep 3
done
MYEOF
```

よくこの記述方法を忘れがちです。
