## [Mac][Homebrew][git] git をインストールする

補完の効く [git](../keyword/git.html) を利用したかったので、[Homebrew](../keyword/homebrew.html) 経由で git をインストールしました。

この手順は [2017-02-19 diary: [Mac][shell] .bashrc を有効にする](http://www.igapyon.jp/igapyon/diary/2017/ig170219.html) かそれと同等を設定を前提とします。

あらかじめ git の補完をインストールします。

```sh
brew install bash-completion
```

.bashrc

```sh
if [ -f $(brew --prefix)/etc/bash_completion ]; then
  source $(brew --prefix)/etc/bash_completion
fi
```

そして git をインストールします。

```sh
brew install git
```
