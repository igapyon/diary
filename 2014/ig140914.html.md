## [Ubuntu]グループにユーザーを追加

[Ubuntu](https://www.ubuntu.com/) でグループにユーザーを追加する方法は以下のとおり。(以下の例は VirtualBox の共有フォルダにユーザーを追加する例)

```sh
sudo gpasswd -a user1 vboxsf
```
