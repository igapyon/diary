## [Python] Python のハローワールド

はじめての Python なのですが、Java にこだわって Jython で体験してみます。

### ソース test.py


```python
# coding: utf-8
print u"Hello."
```



### 実行方法


```sh
jython -C utf-8 test.py
```



### くじけたこと

Windows 上の Jython で日本語を sjis に print できません。。。
