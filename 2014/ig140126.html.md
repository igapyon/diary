## [Shell] [Shell] 引数チェック



### シェルの引数チェックをおこなうサンプル。


```sh
showUsage() {
  cat <<__EOT__ 1>&2
Usage:
  sh ./sample.sh -i 'Hello'
__EOT__
  return 0
}

while getopts i: OPT
do
  case ${OPT} in
    "i" ) argI='true' ; input="${OPTARG}" ;;
  esac
done

if [ ! "${argI}" = "true" ]; then
  showUsage
  exit 101
fi

echo '[i]の引数: '${input} 1>&2
```
