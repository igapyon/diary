## 「いがぴょんの日記ウェブページv3」日記システムの試作中

### おおざっぱ仕様

* Markdown ベースのコンテンツ記述
* 過去の他システム日記データの活用
  * 単純HTML
  * hatena
* github の gh-pages 上での稼働可能性
  * 利用する Markdown を基本的には github の gh-pages 範囲内に抑える
