## [maven] maven インストール


```sh
cd /tmp
tar xvzf apache-maven-3.3.9-bin.tar.gz 
sudo mkdir /opt/apache-maven-3.3.9
sudo mv /tmp/apache-maven-3.3.9/* /opt/apache-maven-3.3.9/
```


/etc/bash.bashrc 

```sh
export MAVEN_HOME=/opt/apache-maven-3.3.9
export PATH=$PATH:$MAVEN_HOME/bin
```
