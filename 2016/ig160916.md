## [Ubuntu] 16.04 に Google Chrome インストールメモ

Ubuntu 16.04.1 に Chrome をインストールした時の手順をメモします。


```sh
sudo dpkg -i google-chrome-stable_current_amd64.deb 
sudo apt install libappindicator1 
sudo apt -f install
```


なぜだか、すんなりとは入りませんでした。
