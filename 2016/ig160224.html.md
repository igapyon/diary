## Windows 用 git クライアント・メモ


### git コマンドの導入


* [http://www.git-scm.com/](http://www.git-scm.com/)
  * Downloads -> Windows
* 右クリック → 管理者として実行
  * Git 2.7.2 64bit
  * 管理者として実行
  * Windows Explorer integration を OFF
  * Use Git from environment
  * Checkout as-is, commit as-is
  * Use MinTTY
