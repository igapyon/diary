## [Ubuntu] Ubuntu 12.04 において、ISO ファイルをコマンドで作成する方法


```sh
mkisofs -l -J -r -o /tmp/test.iso .
```

ロングファイル名 + Joliet + 所有者とモードの設定
なお、この手順だと CD-R のボリュームラベルが固定で CDROM になってしまいます。これを変更するためには -V でボリュームラベルを指定します。
