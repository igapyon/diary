## [JavaFX] MessageBox

JavaFX 2.2 では MessageBox 用の API が存在しません。
追加で [[JfxMessageBox]] という OSS ライブラリを追加することにより、簡単にメッセージボックスを利用することができるようになります。

* [https://ja.osdn.net/proj...gebox/wiki/](https://ja.osdn.net/projects/jfxmessagebox/wiki/)
