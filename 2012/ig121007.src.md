## Ubuntu 12.04 および Debian wheezy では

Ubuntu 12.04 および Debian wheezy では公式パッケージに含まれるとのことです。
☆ただし「Mei の声データ」は含まれないとのことです。


```sh
sudo apt-get install open-jtalk
```
