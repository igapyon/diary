## [Apache][Ubuntu] Apache HTTP メソッド公開状況確認方法


### 利用可能なメソッドの一覧

>|sh|
telnet <ホスト名> 80
OPTIONS / HTTP/1.0

||<


### HEAD を食わせたときの挙動確認

>|sh|
telnet <ホスト名> 80
HEAD / HTTP/1.0

||<


### Ubuntu 12.04 の Apahce2 のデフォルト (抜粋)

Server: Apache/2.2.22 (Ubuntu)
Allow: GET,HEAD,POST,OPTIONS
