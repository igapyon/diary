## [Apache][Ubuntu] Apache HTTP メソッド公開状況確認方法


### 利用可能なメソッドの一覧


```sh
telnet <ホスト名> 80
OPTIONS / HTTP/1.0

```



### HEAD を食わせたときの挙動確認


```sh
telnet <ホスト名> 80
HEAD / HTTP/1.0

```



### Ubuntu 12.04 の Apahce2 のデフォルト (抜粋)


```sh
HTTP/1.1 200 OK
Server: Apache/2.2.22 (Ubuntu)
Allow: GET,HEAD,POST,OPTIONS
```

まあ、Ubuntu のデフォルトは、いちおー安心できそうな雰囲気、、、(知見がもっと溜まらないと判断能力不足...)
