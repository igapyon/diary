## Ubuntu 16.04 に Node.js をインストール

Ubuntu 16.04 に Node.js をインストールする手順を簡単メモ。

### apt ラインに追加
```sh
curl -sL https://deb.nodesource.com/setup_8.x | sudo -E bash -
```

### Node.js のインストール
```sh
sudo apt-get install nodejs
```

### ビルドツールの念押しインストール
```sh
sudo apt-get install -y build-essential
```
