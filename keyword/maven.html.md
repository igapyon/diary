## Maven

Maven は [Apache](apache.html) ファウンデーション配下のビルドシステムです。

### URL

* [https://maven.apache.org/](https://maven.apache.org/)

### 特徴

* Java で書かれた有名なビルドシステムです。
* Java 以外の Java ベースシステムでもこれが利用されます。
* Maven 用のリポジトリが非常に充実しており、これを活用するために Maven を利用する人が多いです。
