## Force.com

[[Force.com]] は [[Salesforce]] を起源とする web ベース統合開発環境です。

### URL

* <@link value="https://www.salesforce.com/products/platform/products/force/" />

### 特徴

* とても生産性高くアプリケーションを開発することができます。
* Classic の場合は Apex や Visualforce というプログラミング言語類を利用して開発します。
