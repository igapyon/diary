## FreeMarker

[[FreeMarker]] は [[Apache]] ファウンデーション配下のテンプレートエンジンです。

### URL

* http://freemarker.org/

### 特徴

* [[Java]] で書かれたテンプレートシステムです。
* HTML 以外に平文テキストも扱うことができます。
* Apache Velocity の後継とみなすこともできます。
* Spring もサポートしている、現時点では最もポピューラーな Java 向けテンプレートエンジンのひとつです。
