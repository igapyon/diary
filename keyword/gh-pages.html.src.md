## gh-pages

[[gh-pages]] は、[[github]] が提供する コンテンツ html 化の機構です。

### URL

* https://pages.github.com/

### 特徴

* markdown (.md) 形式でコンテンツを作成したうえで、[[github]] の `gh-pages` ブランチにコミットすると `ｈttps://username.github.io/reposname` 形式の場所に静的サイトを公開できる仕組みです。
