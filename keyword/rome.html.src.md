## ROME

[[ROME]] は、RSS や Atom のフィードの実現を中心としたライブラリを提供する [[OSS]] です。

### URL

* https://rometools.github.io/rome/

### 特徴

* [[Java]] で書かれた Java 向けのライブラリです。
* RSS や Atom に対応しています。
* 2017/01 時点でもっとも安心して利用可能な OSS ライブラリであると考えました。
