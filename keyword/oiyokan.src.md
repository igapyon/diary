## Oiyokan

[[Oiyokan]] は RDB を OData v4 server (provider) にする SDK です。
Oiyokan を使うと RDBMS を JDBC Driver を使って OData v4 サービスにすることができます。
Apache Olingo / Spring Boot / h2 database ベースです。
