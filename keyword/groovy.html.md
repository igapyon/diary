## Groovy

[Groovy](groovy.html) は [OSS](oss.html) のプログラミング言語のひとつです。

### URL

* [http://groovy-lang.org/](http://groovy-lang.org/)

### 特徴

* [Java](java.html) ベースのプログラミング言語であり、多くの Java ライブラリが利用可能です。
* 簡素なプログラミング記述が可能であり、生産性が高いです。
