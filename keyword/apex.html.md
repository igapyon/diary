## Apex

[Apex](apex.html) は、Salesforce で利用可能なプログラミング言語のひとつです。言語文法は Java 言語に少し似ています。

### URL

* [Apex](https://developer.salesforce.com/docs/atlas.ja-jp.apexcode.meta/apexcode/apex_reference.htm)

### 特徴

* Web ブラウザのみによる開発が可能。
* 言語仕様が Java に似ている。
