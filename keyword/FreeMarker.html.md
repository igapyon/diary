## FreeMarker

FreeMarker は [Apache](https://www.apache.org/) ファウンデーション配下のテンプレートエンジンです。

### URL

* [http://freemarker.org/](http://freemarker.org/)

### 特徴

* Java で書かれたテンプレートシステムです。
* HTML 以外に平文テキストも扱うことができます。
* Velocity の後継とみなすこともできます。
