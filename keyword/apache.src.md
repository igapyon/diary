## Apache

[[Apache]] は、オープンソースプロダクト群を提供している有名なオープンソース集団です。

### URL

* https://www.apache.org/

### 特徴

* さまざまな [[OSS]] のプロダクトを提供するファウンデーションです。
* [[Apache]] は もともとフリーな web サーバーを提供する集団からスタートしました。
