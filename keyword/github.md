## github

[github](github.html) は、無料または有料で利用可能な、ソースコード管理のための有償サービスです。

### URL

* [https://github.com/](https://github.com/)

### 特徴

* とても有名なバージョン管理システムサービスです。
* 多くの [OSS](oss.html) が [github](github.html) 上でホストされています。
